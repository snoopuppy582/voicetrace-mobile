2. 처리하는 정보

VoiceTrace AI는 사용자가 직접 선택한 오디오 또는 비디오 파일을 분석 서버로 전송할 수 있습니다. 서버는 파일에서 오디오를 추출하고 HNR, HF ratio, 클리핑, 무음 비율 같은 품질 지표를 계산합니다.

  • 사용자가 선택한 오디오 파일
  • 사용자가 선택한 비디오 파일
  • 파일명, 파일 형식, 파일 크기
  • 분석 결과 지표와 품질 리포트

앱은 연락처, 위치, 문자, 계정 비밀번호, 사진 라이브러리 전체 접근 권한을 요구하지 않습니다. 사용자가 파일 선택 화면에서 고른 파일만 분석 대상으로 사용합니다.

3. 이용 목적

  • 음성 품질 분석
  • 업로드 준비도 계산
  • 잡음, 고주파 거칠음, 클리핑 등 개선 제안 생성
  • 사용자가 앱 화면에서 분석 결과를 확인하도록 제공

4. 보관 및 삭제

현재 MVP 서버는 업로드 파일을 임시 파일로 처리하고 요청이 끝나면 삭제하는 구조입니다. 앱은 계정 생성 기능을 제공하지 않으며, 사용자를 식별하는 계정 정보를 별도로 저장하지 않습니다. 정식 운영 서버를 제공하는 경우 파일 보관 기간을 최소화하고, 사용자가 삭제를 요청할 수 있는 문의 창구를 제공합니다.

5. 제3자 제공 및 공유

VoiceTrace AI는 사용자의 업로드 파일과 분석 결과를 제3자에게 판매하지 않습니다. 앱 기능 제공을 위해 사용자가 입력한 분석 서버 또는 배포 서버에서 파일 처리가 이루어질 수 있습니다. 클라우드 호스팅, 로그 저장소, 오류 모니터링 도구를 사용하는 경우 해당 서비스 제공자가 처리 위탁자로 포함될 수 있으며, 실제 운영 환경이 확정되면 본 방침에 반영합니다.

6. 보안 처리

정식 배포 서버를 운영하는 경우 HTTPS를 사용해 앱과 서버 사이의 전송을 보호합니다. 로컬 개발 또는 내부 테스트 과정에서는 http://127.0.0.1, http://10.0.2.2 같은 개발용 주소를 사용할 수 있습니다. 사용자는 민감한 개인정보가 포함된 음성 또는 영상 파일을 테스트 서버에 업로드하지 않아야 합니다.

7. 아동 개인정보

VoiceTrace AI는 아동을 대상으로 설계된 앱이 아니며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 중요한 한계

HNR, HF ratio, 스펙트럼 특징은 음성 품질을 설명하는 참고 지표입니다. VoiceTrace AI는 AI 생성 여부를 100% 판정하거나 법적 증거를 제공하는 서비스가 아닙니다.
