# VoiceTrace AI 개인정보처리방침 초안

시행일: 2026-06-19

VoiceTrace AI는 생성형 영상 또는 음성 제작자가 업로드 전 음성 품질을 점검할 수 있도록 돕는 모바일 앱입니다. 이 문서는 Google Play 등록 전 공개 URL로 게시하기 위한 초안입니다. 실제 배포 전에는 운영 서버 주소, 연락처, 보관 기간을 최종 값으로 바꿔야 합니다.

## 수집 또는 처리하는 정보

VoiceTrace AI는 사용자가 직접 선택한 오디오 또는 비디오 파일을 분석 서버로 전송할 수 있습니다. 서버는 파일에서 오디오를 추출하고 HNR, HF ratio, 클리핑, 무음 비율 같은 품질 지표를 계산합니다.

처리될 수 있는 정보:

- 사용자가 선택한 오디오 파일
- 사용자가 선택한 비디오 파일
- 파일명, 파일 형식, 파일 크기
- 분석 결과 지표와 품질 리포트

앱은 연락처, 위치, 문자, 사진 라이브러리 전체, 계정 비밀번호를 요구하지 않습니다.

## 이용 목적

처리한 정보는 다음 목적에만 사용합니다.

- 음성 품질 분석
- 업로드 준비도 계산
- 잡음, 고주파 거칠음, 클리핑 등 개선 제안 생성
- 사용자가 앱 화면에서 분석 결과를 확인하도록 제공

## 보관 및 삭제

현재 MVP 서버는 업로드 파일을 임시 파일로 처리하고 요청 완료 후 삭제하는 구조입니다. 정식 운영 서버에서는 파일 보관 기간을 최소화하고, 사용자가 삭제를 요청할 수 있는 방법을 제공해야 합니다.

## 제3자 제공

VoiceTrace AI는 사용자의 업로드 파일과 분석 결과를 제3자에게 판매하거나 공유하지 않는 것을 원칙으로 합니다. 다만 클라우드 서버, 로그 저장소, 오류 모니터링 도구를 사용할 경우 해당 서비스 제공자가 처리 위탁자로 포함될 수 있으므로, 정식 배포 전 실제 사용 서비스를 기준으로 문서를 갱신해야 합니다.

## 전송 보안

정식 배포에서는 HTTPS를 사용해 앱과 서버 사이의 전송을 보호해야 합니다. 로컬 개발 중에는 `http://127.0.0.1` 또는 `http://10.0.2.2` 같은 개발용 주소를 사용할 수 있습니다.

## 중요한 한계

HNR, HF ratio, 스펙트럼 특징은 음성 품질을 설명하는 참고 지표입니다. VoiceTrace AI는 AI 생성 여부를 100% 판정하거나 법적 증거를 제공하는 서비스가 아닙니다.
