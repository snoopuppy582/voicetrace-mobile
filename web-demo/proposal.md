# VoiceTrace AI - 생성형 영상 음성 품질 분석 리포트

## 한줄 소개

AI 음성 품질 진단

## 기획 배경

생성형 AI 영상은 광고, 숏폼 콘텐츠, 교육 영상, 가상 캐릭터처럼 실감미디어와 가까운 제작 영역에서 빠르게 쓰이고 있다. 화면이 자연스러워도 인물 음성에 미세한 잡음이나 금속성, 부자연스러운 고주파 성분이 섞이면 몰입감은 쉽게 깨진다.

본 프로젝트는 KCC2026 학부생 논문에서 수행한 한국어 AI 동영상 생성 음성과 인간 발화의 차이 분석 경험을 실감미디어 제작 도구로 확장하는 기획이다. 딥페이크 여부를 단정하기보다, 제작자가 업로드 전 음성의 자연스러움과 잡음 리스크를 점검하는 품질 리포트 도구를 지향한다.

## 핵심 기능

1. AI 생성 영상 또는 음성 파일 업로드
2. HNR, HF ratio 기반 음성 품질 분석
3. 인간 발화 기준 샘플과 비교
4. 제작자용 품질 리포트 자동 생성
5. 재생성, 프롬프트 수정, 음성 후처리 필요 여부 제안

## 기술 계획

- 오디오 분석: Python 기반 HNR, HF ratio, 스펙트럼 특징 추출
- 리포트 생성: GPT/Claude 계열 텍스트 생성 모델
- 프론트엔드: 웹 기반 분석 대시보드
- 모바일 앱: React Native/Expo 기반 Android 앱
- 빌드/배포: EAS Build 기반 테스트용 APK 및 Google Play 제출용 AAB
- 확장: 모델별 AI 생성 음성 품질 비교 리포트

## 차별점

단순히 AI 영상을 생성하는 프로젝트가 아니라, 생성형 영상이 실제 콘텐츠로 쓰일 때 몰입감을 깨는 음성 문제를 분석한다는 점이 핵심이다. 실제 연구 경험과 실험 지표를 바탕으로 기획했고, Expo 앱과 Python 분석 서버로 실제 제품에 가까운 구조까지 발전시킬 수 있다.
